\# Country Scraper



A desktop app that searches for country information including capital and population.



\## Features

\- Search countries by name

\- Shows capital city and population

\- Handles invalid country names gracefully



\## How to run

1\. Install dependencies: `pip install requests beautifulsoup4`

2\. Run: `python country_app.py`



\## Built with

\- Python

\- tkinter

\- requests

\- BeautifulSoup
